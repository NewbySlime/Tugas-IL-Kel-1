Detailed Description

Base UI class for handling and showing bar of health. Also it uses effects for "showing" how many the health has decreased by giving a initial health reference that decreased after some delay.

This class uses external component(s);

  • GameObject that has IMaterialReference object.
  • Bound HealthComponent for getting health data.

This class uses following autoload(s);

  • GameHandler for Game events and such.
