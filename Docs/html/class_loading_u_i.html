Detailed Description

UI Class for handling Loading animation based on the supplied AsyncOperation fron any source, but preferably from level loading of Unity. This class uses effect from FadeUI for showing/hiding itself.

This class uses external component(s);

  • Slider for progress bar for loading operations.
