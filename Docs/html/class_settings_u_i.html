Detailed Description

UI Class for handling and giving interaction to modify the Game settings/configuration data. NOTE: this class deattach its save with the one provided by GameHandler by using own PersistanceContext. The save file path can be modified in the editor under PersistanceContext component.

This class uses following component(s);

  • PersistanceContext to use own save file different from the one provided by GameHandler.

This class uses external component(s);

  • AudioMixer the target mixer to modify the audio settings.
  • Slider interface interaction for modifiying settings.
  • ButtonBaseUI as the "back" button to close this UI.

This class uses autoload(s);

  • GameHandler for game events and such.
  • InputFocusContext for asking focus to use input.
