Detailed Description

Component as a functionality part of "Item" mechanic where this object becomes a collectible of an item associated with it. An object with InventoryData can collect this item object.

This class uses external component(s);

  • RigidbodyMessageRelay for body interaction events.
  • SpriteRenderer for displaying item's image.

This class uses autoload(s);

  • ItemDatabase for getting data about certain item.
