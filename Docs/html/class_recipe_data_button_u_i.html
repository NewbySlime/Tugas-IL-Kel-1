Detailed Description

Button for showing bound recipe data and handling the interactions. This class also used by RecipeBookUI.

This class uses external component(s);

  • Unity's MaskableGraphics for showing the resulting item image from a bound recipe.
  • Unity's TMP Text UI for presenting the data related to the recipe.

This class uses autoload(s);

  • ItemDatabase for getting data of items.
