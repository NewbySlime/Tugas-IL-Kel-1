Detailed Description

OUTDATED This UI class handles button interaction with using certain item to bind and be associated with.

This class uses external component(s);

  • Target Unity's MaskableGraphics for displaying the Image of the associated item.
  • Unity's TMP Text UI for displaying item count for the associated item.
