Detailed Description

Class for handling lighting object used for Game world's light. This handler class also changes the configuration of the light based on the state of the Game (ex. the time period handled by GameTimeHandler).

This class uses following component(s);

  • Light2D the light object used as the Game world's light.

This class uses autoload(s);

  • GameHandler for game events and such.
  • GameTimeHandler for configuring Game's world time and watching its events.
