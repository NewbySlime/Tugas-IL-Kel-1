Detailed Description

Class for handling the Game's time also handling scenario for period of time. Period of time defined in this class; (period type | Scenario ID used)

  • Daytime | "daytime_scenario"
  • Nighttime | "nighttime_scenario"

This class uses autoload(s);

  • GameHandler for using scenario used by this class for changing period of time.
