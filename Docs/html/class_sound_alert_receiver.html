Detailed Description

Class to "receive sound" transmitted by SoundAlertTransceiver. Another component/object can use the events use in this class to receive the "sound" transmitted from a source. For further explanation, see Reference/Diagrams/SoundAlert.drawio

NOTE: this will not actually capture audio from the Unity's Sound system, instead it uses "imaginary sound" in a form of area of effect to trigger "hearing" events.
