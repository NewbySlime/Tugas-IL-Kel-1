Detailed Description

Class for spawing any objects (stored as inactive object in certain container) whenever it got triggered. The class also handles auto spawning with some conditions that can be modified with available variables.

This class uses external component(s);

  • GameObjects as containers that holds objects used in this class.
  • Target GameObject(s) for condition checking (minimal distance from the spawner for auto spawn to be triggered).
