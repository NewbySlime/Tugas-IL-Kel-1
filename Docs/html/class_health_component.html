Detailed Description

Base component for Game's entities to handle health system for the entity. When the entity is "dead" the entity will be despawned, but can be opted to not despawned for reasons like presentation of the story.

This class uses external component(s);

  • Rigidbody2D as the base body of the component.
  • Animator for animation handling for effects used in this component.
  • AudioCollectionHandler for playing audio of certain ID used as the effects in this component.

For another effects that can be used alongside this component;

  • DamagedEffect
  • HealedEffect
  • RecoveryEffect
  • VulnerableEffect
