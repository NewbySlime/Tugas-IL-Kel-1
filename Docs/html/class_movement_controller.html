Detailed Description

Component for handling movement system used in the Game. For further explanation, see Reference/Diagrams/MovementController.drawio

IMPORTANT This class needs child objects to be functional;

  • "GroundCheck" physics body (trigger) that has RigidbodyMessageRelay to let this class know that this object has touched a ground.
  • (Optional) "WallHugRight" and "WallHugLeft" physics body (trigger) that has RigidbodyMessageRelay to let this class know that this object has touched a wall (that can be used for wall-jumping).

This class uses following component(s);

  • Rigidbody2D physics body for this class.
  • Collider2D (any shape) collider component for the physics body.

This class uses external component(s);

  • RigidbodyMessageRelay for getting physics event for another detached part of the body.
  • Unity's Animator component for handling movement animation.
  • AudioCollectionHandler for playing certain audio used in the animation.
