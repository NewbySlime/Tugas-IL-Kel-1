Detailed Description

Class to "transmit sound" that can be received by SoundAlertReceiver. This class is used to create an "imaginary sound" using an area of effect to accomodate Unity's Sound system for a Game object's ability to "hear" a sound that can be used to trigger a flag in the object. For further explanation, see Reference/Diagrams/SoundAlert.drawio

This class uses following component(s);

  • Unity's CircleCollider2D physics body (as trigger) for the area of effect to capture nearby SoundAlertReceiver objects.
