Detailed Description

Class for handling data, storing and loading the data with the OS' filesystem. The path for save file can be readjusted using _SaveFileLocation. For any objects that wanted to store or load the data, use PersistanceContext.IPersistance and also use the events used in this class. For further explanation, see Reference/Diagrams/SaveSystem.drawio
