Detailed Description

UI element for showing and giving "focus" effect/animation to fullbody image of a character that used in DialogueCharacterUI. Image for the characters are fetched data from CharacterDatabase.

This class uses following component(s);

  • RectTransform for base UI's needed components.

This class uses external component(s);

  • Image UI for storing character's image.

This class uses following autoload(s);

  • CharacterDatabase for getting character data about the bound character.
