Detailed Description

A class that handles interaction event with dialogue bubble. See InteractionHandler for interaction handling.

This class uses Component(s);

  • DialogueUI in DialogueBubblePrefab for Dialogue presentation handling.
  • ShrinkUI in DialogueBubblePrefab for "show" and "hide" effect to the UI.

Interlinked Component(s);

  • InteractionHandler uses the class events for processing interaction for Dialogue.
