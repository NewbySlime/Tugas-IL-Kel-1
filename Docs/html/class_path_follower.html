Detailed Description

Component for controlling MovementController autonomously using pathfinding in 2D space. For further explanation, see Reference/Diagrams/AIMovement.drawio

This class uses following component(s);

  • Arongranberg's Seeker Pathfinding object for processing a graph of path to a target position.
  • MovementController for movement used in the Game.

Thanks to Arongranberg's Pathfinding "A* Project" library to make this possible.
