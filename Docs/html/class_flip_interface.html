Detailed Description

Class used for handling flipping an object (in 2D space) or changing the direction of the object(s) horizontally or vertically. List compatible object(s) to flip;.

  • SpriteRenderer
  • FlipInterface (Flipping itself is not possible)

For objects that is not compatible, any object can catch the message interface used by this class by using FlipInterface_SetFlippedX(bool) and/or FlipInterface_SetFlippedY(bool).
