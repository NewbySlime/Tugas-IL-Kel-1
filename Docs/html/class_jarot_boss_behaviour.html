Detailed Description

Enamy Behaviour controller for Jarot. For behaviour diagram, look at Reference/Diagrams/JarotBossBehaviour.drawio.

This behaviour uses following component(s);

  • MovementController to let the behaviour have control over the object movement.
  • to give the object ability to have vitality features. as an extension of the behaviour used as a AI follow helper.
  • VulnerableEffect used as a vulnerable time window after power striking.

This behaviour uses external component(s);

  • DamagerComponent collider object that seperate from the actual object that acts as a Damager.
  • Animator Unity animation handler for certain object state.
  • DEBUG_TARGET [DEBUG_MODE] target object to follow once the Game started.
