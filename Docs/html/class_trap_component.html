Detailed Description

Helper class for DamagerComponent to make it as a static damager (not moving). As this component does not use damager data from an item, the class uses a determined data from DamagerComponent.DamagerContext and DamagerComponent.DamagerData as the configuration used in this class.

This class uses external component(s);

  • DamagerComponent the damager used for this component.

This class uses autoload(s);

  • GameHandler for game events and such.
