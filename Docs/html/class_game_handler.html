Detailed Description

Class for handling Game's functions, events, and other objects/components that serves as the internal functions.

This class uses following component(s);

  • PersistanceContext for handling file save for a game state in certain time.

This class uses external component(s);

  • Object for containing ScenarioHandlerVS.
  • LevelCheckpointDatabase for storing data about all checkpoint in a scene.
  • GameTimeHandler for setting up the Game's time.
  • ScenarioDiagramVS for handling Game's scenario or parts of the Game's story.
  • GameUIHandler for getting all UIs in the Game.
  • InputFocusContext for handling the Game's input.
  • GameRuntimeData for storing data about the Game in runtime for use inbetween scenes.
  • GameOverUI
  • FadeUI UI for covering up the screen.
  • LoadingUI
See also
GameUIHandler, GameTimeHandler, GameRuntimeData
