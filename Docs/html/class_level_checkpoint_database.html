Detailed Description

Database for storing data about CheckpointHandler in a certain scene. This database have to be updated whenever a scene has been changed due to this class does not use nor store any data related to a scene. Instead, it will get all CheckpointHandler object existed in the scene. This database should exist only once throughout the Game instance.
