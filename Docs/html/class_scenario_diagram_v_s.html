Detailed Description

Class for handling Scenarios. This class acts as a server that holds a list of ScenarioHandlerVS. The difference between ScenarioHandlerVS is that this class handles a collection of scenarios contained while ScenarioHandlerVS handles a collection of subscenarios contained in a scenario. For further explanation on how scenario works in the Game, read the diagram contained in Reference/Diagrams/Scenario.drawio

This class uses prefab(s);

  • Prefab that has ScenarioHandlerVS for prepping scenario from scenario data.
