Detailed Description

Class to handler Minigame system for when Enemy Battles in the Game. In this class, there are conditions as an outcomes;.

  • Win condition is based on if all the "enemies" are defeated.
  • Defeat condition is based on if all the "allies" are defeated.

NOTE: bound objects should have HealthComponent and ObjectFriendlyHandler for the Minigame to work properly.

Optional Component(s);

  • ObjectSpawner Spawn Object(s) that then be bound to the class based on ObjectFriendlyHandler.
