Detailed Description

Class that will handle both multi-character dialogue by also showing which character is talking.

This class uses external component(s);

  • DialogueUI base class used for handling dialogues.
  • Unity's TMP Text UI for displaying character names.

This class uses prefab(s);

  • Prefab for showing characters image using CharacterFocusUI.
  • Prefab for handling sequencing (SequenceHandlerVS).

This class uses following autoload(s);

  • CharacterDatabase for getting character data.
  • InputFocusContext for asking focus for using input.
