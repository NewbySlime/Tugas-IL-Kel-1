Detailed Description

TODO: recheck for next feature.

UI Class for visualizing QTE system.

This class uses external component(s);

  • Slider for the QTE response "bar".
  • Unity's MaskableGraphics object for presenting the symbol used.
  • GameObject that has RectTransform and IMaterialReference for "Accept" timing bar to resize and manipulate its color.
