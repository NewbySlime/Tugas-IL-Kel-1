Detailed Description

Class for handling interaction with other objects that has InteractableInterface. This class acts as the actor, all events surrounding interactions are handled here. For further explanation, see Reference/Diagrams/InteractionHandling.drawio

This class uses external component(s);

  • RigidbodyMessageRelay for getting physics events of the target body used.
  • InteractableInterface for signalling the interface for interaction events.
