Detailed Description

Class that triggers a sequence contained in this class by using event from Collider2D (as trigger) for when an object entered its physics body. Uses AreaTriggerActuator to differentiate which object that can be use as a source of triggering.

This class uses external component(s);

  • SequenceHandlerVS for sequencing system used in this class.
  • RigidbodyMessageRelay for handling Collider2D's Unity event.

This class uses autoload(s);

  • GameHandler for game events and such.
