Detailed Description

UI Component for showing Game Over screen.

This class uses external component(s);

  • Unity's TMP Text UI for showing different types of game over causes.
  • ButtonBaseUI for giving options to the player after game over state.

This class uses following autoload(s);

  • GameHandler for using Game events and such.
