Detailed Description

Class for giving an object a functionality to get "alerted" when a valid object is entering its view. An object can be valid when it is allowed to be collided or entered in this Collider2D area. For further explanation, see the diagram contained in Reference/Diagrams/AlertTrigger.drawio

This class uses following component(s);

  • CircleCollider2D trigger collider for watching objects entering or exiting.
