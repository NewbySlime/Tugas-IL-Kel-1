Detailed Description

Class for modifying components related to Character object presentation (like Unity's Animator component) based on the ID of a target character.

This class uses external component(s); Unity's Animator component to modify its animation data to fit the target character.

This class uses autoload(s);

  • CharacterDatabase for getting target Character data.
