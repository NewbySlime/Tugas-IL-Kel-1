Detailed Description

Class for handling a list of subscenario contained in this class. For a class that handles sets of scenarios, see ScenarioDiagramVS. For further explanation on how scenario works in the Game, read the diagram contained in Reference/Diagrams/Scenario.drawio

This class uses prefab(s);

  • Prefab that has QuestHandlerVS for quest handling in a scenario.
  • Prefab that has SequenceHandlerVS for sequence handling in a scenario.
