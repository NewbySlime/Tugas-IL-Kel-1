Detailed Description

Component that gives an object functionality to become an object that can damage another object that has HealthComponent. This class is handled by WeaponHandler or can be entirely independent (as an example TrapComponent).

This class uses external component(s);

  • SpriteRenderer for displaying sprite for the damager/projectile (optional if displayed in another object).
  • SoundAlertTransceiver for alerting any object using "sound" range.
  • AudioSource for playing a sound when the object is interacted.
