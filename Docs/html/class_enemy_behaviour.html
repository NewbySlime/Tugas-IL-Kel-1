Detailed Description

Base behaviour class for enemy. For behaviour diagram, look at Reference/Diagrams/EnemyBehaviour.drawio.

This behaviour uses following object(s);

  • FollowerBehaviour for following and targeting its enemy.
  • NPCRandomBehaviour for letting the object wander around aimlessly, waiting for an enemy to cross its path.
  • PathFollower component for pathfinding and AI movement.

This behaviour uses external component(s);

  • AlertTrigger for getting any enemy on its view.
  • DamagerComponent as a melee object used by this enemy.
