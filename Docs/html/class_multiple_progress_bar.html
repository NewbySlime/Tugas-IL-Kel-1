Detailed Description

Class for handling multiple progress bar with using one value. Brief explanation: the progress uses non-normalized value, and the value is used for counting how many progress bar to use. (ex. Expected value 8.00 which means 8 progress bar to be visualized).

This class uses external component(s);

  • GridLayoutGroup for arranging each ProgressionTexture to its cells.

This class uses prefab(s);

  • Prefab that has ProgressTexture as the "progression" effect for each bar.
