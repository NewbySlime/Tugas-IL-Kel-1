Detailed Description

Component for handling weapon and gives its wielder (such as PlayerController) Game interactions for combat system. The weapon bound in this component is based on the bound item that has data about WeaponItem. For ammo count handling, it is controlled by the wielder due to some wielder such as enemies might use the weapon with indefinite ammot count.

This class uses autoload(s);

  • ItemDatabase for getting data about certain item.
